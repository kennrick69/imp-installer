            Preciso de administrador
          
              
        
              
        
              
          
              
            O próximo passo configura o WSL2 no Windows.
            O sistema só permite isso com privilégio de administrador —
            é a mesma proteção que aparece quando você instala um programa novo.
          
              
          
              
            
              O que vai acontecer ao clicar "Reabrir como administrador":
              
            
                
              
              1. O Windows vai mostrar uma janela do Controle de Conta de Usuário (UAC)
                2. 
              
              2. Você clica em "Sim"
                3. 
              
              3. O instalador reabre automaticamente, agora com permissão
                4. 
              
              4. Sua progressão continua de onde estava (não precisa começar de novo)
                5. 
            
              
          
              
          
              
            Se preferir fazer manualmente: feche este instalador, clique com botão direito
            no .exe e escolha "Executar como administrador".
